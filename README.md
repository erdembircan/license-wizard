# License Wizard

A tool that helps developers choose and apply the right software license to their projects.

## Overview

License Wizard guides you through selecting an appropriate open-source license based on your needs, and handles the creation of properly formatted license files.

## Knowledge Base

The `docs/knowledge-base/` directory contains reference material on:

- Copyright fundamentals and common mistakes
- License types (permissive vs. copyleft)
- SPDX standard identifiers and expressions
- License file conventions and formatting
- Package manager integration
- Canonical license text sources

See [mind-map.md](docs/knowledge-base/mind-map.md) for a visual overview of all topics.
